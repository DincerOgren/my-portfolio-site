Overview

Night Shift is a 3D top-down action RPG built upon systems inspired by GameDev.TV’s RPG course. In this solo development project, I designed and implemented multiple core systems including point-and-click movement, dynamic enemy AI, a robust inventory and equipment framework, and scalable combat mechanics. The project focuses on providing fluid gameplay, intuitive controls, and a strong foundation for stat-based progression and wave-based survival mechanics.




Technical Details

  • Movement System: Implemented a point-and-click movement system using Unity’s NavMeshAgent. Players can navigate by clicking on terrain, offering intuitive and responsive control.
  • Contextual Cursor System: Developed a custom cursor handler that dynamically switches the cursor sprite based on context — including UI hover, interactable objects, and enemies — improving player feedback and interaction clarity.
  • Inventory System: Created a modular inventory system that supports storing, equipping, and using various item types. Items are categorized into usable (e.g., potions) and equipable (e.g., armor, weapons).
  • Equip Slots: Players can assign gear to specific slots. Equipping updates the character’s attributes in real-time and affects gameplay such as defense or damage output.
  • Weapon System: Designed a flexible weapon framework using Scriptable Objects. Each weapon includes custom attributes (damage, attack range, icon, animation override, prefab) and is easily extendable.
  • Enemy AI: Enemies patrol and detect the player within a defined radius. Once aggroed, they chase and attack, triggering nearby enemies as well. Attack types vary (melee or ranged) and use animation overrides.
  • Stat and Leveling System: Built a scalable stat and level system for both players and enemies. Stats such as health, damage, and XP scale dynamically with level.
  • Wave System: Introduced a wave-based spawning system that increases game difficulty over time by upgrading enemy levels and types with each completed wave.
  • Performance Optimization: To ensure smooth performance, I implemented an object visibility system that disables off-screen objects and reduces light intensity for unused light sources.

Technologies Used

  • Game Engine: Unity
  • Programming Language: C#
  • Development Tools: Visual Studio, GitHub
  • UI Framework: Unity UI & TextMeshPro
