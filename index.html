Who Am I ?

About Me

Ever since I wrote my first line of code, I've been hooked on making games. I enjoy bringing worlds to life through Unity and C#, always aiming to create experiences that are both enjoyable and unique.
I love the entire process—from designing mechanics and writing scripts to solving unexpected bugs. Every project I work on is a new adventure, and I'm always looking for ways to learn and grow as a developer. My goal is to create games that not only entertain but also leave a lasting impression

What I Did ?

Portfolio

Poly Surf

Poly Surf is a fast-paced endless runner game built with Unity. Dodge obstacles, collect power-ups, and aim for the highest score in a vibrant low-poly world.

Details

Match3-Prototype

This Match-3 prototype focuses on the core game mechanic: swapping adjacent tiles to create matches of three or more. Built with Unity, it's a simple and clean implementation aimed at testing the base functionality without additional features or polish.

Details

Fruit Party

Fruit Party is a Unity-based slot game prototype inspired by vibrant fruit-themed casino games. It features cascading reels, colorful visuals, and a simple match-based win system. Designed as a base project, this prototype focuses on the core mechanics of slot gameplay, offering a foundation for future enhancements like multipliers, bonuses, and UI/UX improvements.

Detaylar

Charm Holder

Developed during the Arnavutköy Game Development Bootcamp with an 8-person team, Charm Holder is a first-person shooter where players wield elemental rings to cast diverse magical abilities. My contributions focused on designing core gameplay systems such as ring selection, spell casting with Scriptable Objects, and special attacks. I also implemented enemy AI, combat mechanics, and player health features to bring the combat loop to life.

Details

Night Shift

Night Shift is a 3D top-down action game inspired by GameDev.TV’s RPG course. I built key systems such as point-and-click movement, a dynamic cursor system, and an expandable inventory with usable and equipable items. Additionally, I developed modular combat mechanics, AI behavior, a wave-based enemy spawning system, and performance optimizations — all crafted to support scalable gameplay and smooth player experience.
