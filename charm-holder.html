Overview

Enigmat is a first-person magical combat prototype developed during a week-long bootcamp. As the team leader, I was responsible for core gameplay mechanics, combat systems, and integration tasks. The main goal was to explore dynamic spellcasting mechanics and enemy encounters using a modular, scalable architecture. Despite limited time and resources, we successfully delivered a working prototype featuring diverse elemental abilities, enemy AI, and responsive player feedback systems.




Technical Details

  • Elemental Ring System: Implemented a ring selection mechanic allowing players to switch between different elemental types (fire, ice, lightning). Each ring dynamically updated both gameplay logic and the UI, ensuring a seamless player experience.
  • Modular Spell System (ScriptableObjects): Designed a highly extensible spell system using Unity’s ScriptableObject architecture. Each spell encapsulated data like damage, cooldown, visual effects, and behaviors, enabling future expansion with minimal code changes.
  • Enemy AI: Developed basic yet functional enemy behaviors for two types: mages, who cast ranged spells at the player, and skeletons, who engage in close combat. Both types use detection radius to switch between patrol, chase, and attack states based on player proximity.
  • Elemental Projectile System: Developed core ranged combat mechanics. Based on the selected ring, players could launch element-specific projectiles with different effects (e.g., burn, freeze, shock). Projectiles were handled with modular prefabs for easy tuning.
  • Special Ability Integration: Integrated a sky-based special ability system in collaboration with teammates. Using an indicator system, players could target zones to summon environmental strikes like lightning bolts, meteors, or icicles.
  • Health System: Created a health system for the player, including a visual health bar that accurately reflects the player’s health status, providing clear feedback during gameplay.
  • Animation Integration:Ensured all combat, movement, and spellcasting animations worked smoothly. Integrated Unity’s Animator Controller with blend trees and trigger parameters to connect gameplay logic with visual feedback.

Technologies Used

  • Game Engine: Unity
  • Programming Language: C#
  • Development Tools: Visual Studio, GitHub
  • UI Framework: Unity UI & TextMeshPro
