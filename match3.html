Match3 - Prototype

Overview

​In this Match-3 game prototype, I focused on developing a responsive and efficient core game mechanic. The project emphasizes smooth tile swapping, accurate matching detection, and dynamic board refilling to ensure an engaging user experience. Although not released as a full game, this prototype serves as a foundation for potential expansion into a complete product.



Technical Details

  • Grid Management: Implemented a flexible grid system that initializes and manages the game board, allowing for easy adjustments to dimensions and tile types.
  • Tile Behavior: Developed individual tile components responsible for handling user input, animations, and state changes, ensuring seamless interactions and visual feedback.
  • Match Detection: Created algorithms to identify horizontal and vertical matches of three or more tiles, triggering appropriate effects and point calculations.​
  • Board Refilling: Designed a system to collapse matched tiles and refill empty spaces with new tiles from above, maintaining continuous gameplay.
  • Game Management: Centralized control flow through a game manager that oversees the game state and score tracking.

Technologies Used

  • Game Engine: Unity
  • Programming Language: C#
  • Development Tools: Visual Studio, Git
  • UI Framework: Unity UI & TextMeshPro
